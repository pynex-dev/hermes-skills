# Hermes Skills Tap

Community skills for Hermes Agent by [pynex-dev](https://github.com/pynex-dev).

## Install

Add this tap to Hermes:

```bash
hermes skills tap add pynex-dev/hermes-skills
```

Then browse or install:

```bash
hermes skills search kagi --source pynex-dev/hermes-skills
hermes skills install pynex-dev/hermes-skills/kagi-search
```

## Skills

| Skill | Description |
|-------|-------------|
| **kagi-search** | Kagi Search MCP server — high-quality, ad-free web search with multiple workflows (general, news, videos, podcasts, images) and page extraction |
